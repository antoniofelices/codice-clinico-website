# Códice Clínico website

## Documentation

### Installation

You can use this project as a template — simply modify the files inside the 'content' folder.

1. Clone this repo

```bash
$ git clone git@github.com:antoniofelices/codice-clinico-website.git .
```

2. Install packages

```bash
$ npm install
```

3. Run develop

```bash
$ npm run dev
```
